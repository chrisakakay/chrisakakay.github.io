Project: OpenRemoteServer

It's a SublimeText3 plugin (maybe works with ST2 too).
Made to make testing and file handling easier (from browsers)

The plugin itself starts a HTTP server in ST3 and accepts the following query:
http://localhost:3333/?file=\file.txt:41 Which will open up the file if it exists, and goes to the line given in the parameters.
