So lets start with HTML

HTML is your entry to make a website. For example when you open an url in you browser the standard page will be your index.html which is the main page of a static website. Of course it depends on the server which hosts the website, but this is accurate most of the time. The current version of HTML is 5.

The HTML file will be parsed by your browser. If it contains CSS or JS file references, the browser will load those too. The HTML will define the structure of your site and you can design it with CSS. You

This is a basic example: 


    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <p>Hello world!</p>
        </body>
    </html>

Try it! Save it as index.html (or anything you want) and open it in your browser. You will see a black "Hello world!" text. It's not much, but still it's progress. :)

So how does it work?

In HTML you use tags. Everything (this is not exactly true, there are special cases, but almost everything) has an opening tag and a closing tag with a /. For example: <head> ... </head>

The basic example in details:


    <!DOCTYPE html>                       This will tell the browser how to parse your page
    <html>                                Opening html tag
        <head>                            Opening head tag
            <title>Page Title</title>     Page title, browser tabs will show this
        </head>                           Closing head tag
        <body>                            Opening body tag
            <p>Hello world!</p>           Simple paragraph
        </body>                           Closing body tag
    </html>                               Closing html tag

You can (and should) use self closing tags aswell, but not every tag can be used as a self closing tag.
Consider these: <img src="" /> In this case the image itself is not containing any other tag (at it never will) so it is a self closing tag. But a DIV is a container element and even if its not containing any other tag, it can not be used as a self closing tag.

In the example almost all tags are needed, except the P with the "Hello world!" text. The HTML tag should contain the head and the body. The head can contain references to CSS and JS files, this way the browser will load these first.

NOTE: JS files should not be included in the head, because they will block the rendering chain, but the CSS files should be so when the browser renders our page we will have the styles for the rendering. References can be included in the body aswell, JS references should be included at the end of the body, just before the body closing tag.

The head won't be rendered in any way, but the body will, so thats where we put our nasty stuff.
